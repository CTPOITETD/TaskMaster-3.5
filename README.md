# Task Master
1. ABOUT:

This is a task management tool designed for creating and sharing 
structured task lists. 
The entire package (including .py, .bat, and .ico) is ultra-lightweight, 
clocking in at approximately 20KB.

2. GETTING STARTED:
- Run "TaskMaster.py" first to ensure the environment is ready.
- Right-click "TEST.task" -> "Open with..." -> Scroll to the bottom 
- Select "Look for another app on this PC" -> Choose "Launcher.bat"
- Enable "Always use this app to open .task files"
- Try to double-click "TEST.task". 
  If it loads your tasks — success! If not... try again.

3. UI OVERVIEW:
The interface is packed with features. Here is what they do:

	[ FILE ]
- New Window: Opens a separate instance (independent from the first one).
- Open .task: Loads any compatible .task data file.
- Save .task: Exports your current list into a file.
- Clear All: Wipes the screen. (Warning: This doesn't mean your real-life 
             tasks are gone, you still have to do them!)

	[ EDIT ]
- Create Task: Adds a new entry (Shortcut: + CREATE button).
- Replace: Swaps the selected task's text with your new input.
- Delete Selected: Removes the highlight. No "Ctrl+Z" here, so be careful!
- Select All: Highlights every task in the current list.

	[ VIEW ]
- Toggle Theme: Switches between Dark (Black) and Light (White) modes.
- Language: Toggle between English and Ukrainian. (Yes, only 2 for now...)

	[ TIMER ]
- Start/Stop/Reset: Professional-grade tracking for your productivity.

	[ HELP ]
- Hotkeys: Quick reference for power users.
- Info: System details and credits.

	[ OPEN PROGRAM ]
- A built-in shortcut to launch any external executable or file.
